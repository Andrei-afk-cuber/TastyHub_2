# Tasty Hub
## Description
Tasty Hub - это серверная часть платформы для обмена рецептами. Она необходима для работы приложения. Она обеспечивает основную бизнес-логику и позволяет клиенту взаимодействовать с базой данных. 
## Installation
The server can be installed from this repository:
```commandline
git clone https://github.com/Andrei-afk-cuber/TastyHub-Server.git
```
The client can be installed:
```commandline
git clone https://github.com/Andrei-afk-cuber/TastyHub-Version-2.git
```
For install all libraries:
```commandline
pip install -r requirements.txt
```
